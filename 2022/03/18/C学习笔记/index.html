1数据类型

short int 短整型
int 整型
long int 长整型
long long 更长整型
float 单精度浮点数
double 双精度浮点数

2变量、常量

常量

变量定义 数据类型+名+值
全局变量 定义在代码块外部 作用域为整个工程 extern声明外部符号
局部变量 定义在代码块内部
局部变量与全局变量同名时，局部变量优先
C语言定义变量应在程序开始
